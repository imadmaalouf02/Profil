# Portfolio - Imad Maalouf

Portfolio professionnel d'Imad Maalouf, Doctorant en Intelligence Artificielle, Data Science et Digital Twin pour l'optimisation de l'energie solaire.

## Structure du Projet

```
portfolio-pro/
├── index.html              # Redirection vers templates/
├── templates/
│   └── index.html          # Page principale HTML
├── css/
│   ├── main.css           # Styles principaux + Responsive
│   └── animations.css     # Animations avancees
├── js/
│   ├── main.js            # Navigation, Theme, Filtres
│   └── animations.js      # Scroll reveal, effets speciaux
├── assets/
│   ├── images/            # Photos et images de demo
│   └── videos/            # Videos de demonstration
└── README.md
```

## Fonctionnalites

- **Design Moderne** : Interface epuree avec animations fluides
- **Theme Clair/Sombre** : Basculement automatique avec sauvegarde
- **Animations** : Reveal au scroll, particules, effets hover
- **Responsive** : Adaptation mobile, tablette et desktop
- **Sections Completes** :
  - Hero avec photo et statistiques animees
  - A propos avec informations personnelles
  - Recherche (Digital Twin & Energie Solaire)
  - Competences techniques avec barres de progression
  - Experience professionnelle (timeline)
  - Projets avec filtrage par categorie
  - Publications academiques
  - Certifications
  - Formulaire de contact

## Technologies Utilisees

- HTML5 semantique
- CSS3 avec variables CSS
- JavaScript vanilla (ES6+)
- Font Awesome pour les icones
- Google Fonts (Inter, JetBrains Mono)
